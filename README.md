## 🚀 Deploy

O site está configurado para deploy automático no **Netlify**.

### Como fazer deploy:

1. **Conecte o repositório ao Netlify:**
   - Acesse [netlify.com](https://netlify.com)
   - Clique em "New site from Git"
   - Selecione GitHub e autorize
   - Escolha o repositório `CrewsMagzOfc`

2. **Configuração automática:**
   - Netlify lerá o arquivo `netlify.toml`
   - Qualquer push para `main` fará deploy automático

3. **Seu site ficará em:**
   - `https://[nome-aleatorio].netlify.app` (temporário)
   - Configure domínio personalizado depois em Netlify → Domain settings

---

## 📝 Como fazer mudanças

1. Edite os arquivos `.html`, `.css` ou `.js`
2. Faça commit e push para `main`
3. Netlify detecta e faz deploy automaticamente
4. Site atualiza em segundos
